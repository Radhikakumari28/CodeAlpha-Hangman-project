# Hangman Game

A simple Hangman game developed using Python.

## Features
- Random word selection
- Limited attempts
- Shows guessed letters
- Input validation
- Play again option

## Technologies Used
- Python

## How to Run
1. Download the project files.
2. Open terminal in the project folder.
3. Run:

python hangman.py
